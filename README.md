# Flash_todo

# A Todo List demo App
This is a todo list demo designed for Bhavin Sen. 
```

## Getting started
```bash
git clone https://github.com/bhavinsen01/Flash_todo.git
cd Flash_todo
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
export FLASK_APP = app
run python3 run.py
```
